## 📋 Overview

Kamdi Market is a **government-approved enterprise marketplace** specializing in cybersecurity tools, ethical hacking software, enterprise hardware, and certified devices. Built with Next.js 15+, TypeScript, and Tailwind CSS.

**Live Demo:** [https://market.kamdi-dev.click](https://market.kamdi-dev.click)

## ✨ Features

### Core Features
- 🎨 **Dark/Light Mode** - Persistent theme with system preference detection
- 🔒 **Enterprise-Grade Security** - Production-ready authentication ready
- 📱 **Fully Responsive** - Works on all devices
- ⚡ **Server-Side Rendering** - Optimized performance with Next.js 15
- 🔍 **SEO Optimized** - Complete meta tags, sitemap, and robots.txt
- 📦 **Product Catalog** - Structured for tech and cybersecurity products

### SEO & Metadata
- Dynamic Open Graph tags
- Twitter Card support
- JSON-LD structured data (Product Schema)
- Automatic sitemap generation
- Robots.txt configuration
- Canonical URLs
- Multi-language alternates

### Technical Stack
| Technology | Version | Purpose |
